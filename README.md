# LINUXtips-PICK
Projeto Prático.

 A Linuxtips fez a criação de uma aplicação de gestão de senhas, e a ideia é subir isso para uma imagem de container e fazer deploy no kubernetes e automação e monitoramento contínuo.

 
 
 Tecnologias usadas:
 Docker, 
 Kubernetes,
 Helm
 Kyverno
 Cosign
 Prometheus
 apko e Melange para garantir a segurança
 automação e monitoramento contínuo da aplicação.
